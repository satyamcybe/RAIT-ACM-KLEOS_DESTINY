# Contributing to Pramaan

First off, thank you for considering contributing to Pramaan! It's people like you that make Pramaan such a great tool for financial inclusion of the gig economy.

## How to Contribute

### 1. Reporting Bugs
- Ensure the bug was not already reported by searching on GitHub under Issues.
- If you're unable to find an open issue addressing the problem, open a new one. Be sure to include a title and clear description, as much relevant information as possible, and a code sample or an executable test case demonstrating the expected behavior that is not occurring.

### 2. Suggesting Enhancements
- Open a new issue with a clear title and description.
- Explain why this enhancement would be useful to most users.
- If you can, provide a mockup or architectural proposal.

### 3. Pull Requests
- Fork the repository and create your branch from `main`.
- If you've added code that should be tested, add tests.
- If you've changed APIs, update the documentation.
- Ensure the test suite passes (`npm test`).
- Issue that pull request!

## Code Style
- We use TypeScript and ESLint. Please ensure your code passes `npm run lint`.
- Follow the existing folder structure (Layer 1, Layer 2, Layer 3 logic in `src/lib/intelligence`).

Thank you for helping us build trust!
